# Smart Toolkit 1.0.0

Initial public release of Smart Toolkit, an Android utility app that brings 21 everyday tools into one local-first app.

## Highlights

- 21 utilities across productivity, measurement, device info, camera, and conversion workflows
- Jetpack Compose + Material 3 UI with favorites, search, themes, and an in-app user guide
- Local persistence with Room and DataStore for notes, settings, favorites, and utility history
- Notification controls for timer, stopwatch, flashlight, and tally counter
